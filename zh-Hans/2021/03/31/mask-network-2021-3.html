🌳 生态系统建设

- NFT

🧩 Mask Network x DEGO Finance｜共建 Web 3.0 时代的 NFT 入口

Mask Network 与跨链 NFT 和 DeFi 协议 DEGO Finance 达成合作，双方计划就 NFT 盲盒，游戏，社交进行一步的合作，实现用户在 Twitter 进行一键 NFT 创造，收藏与买卖。

🔓 Mask Network x Cent｜在 Twitter 上买你刚刷到的推

Mask Network 和数字身份平台 Cent 展开合作并发布联合赏金，目标使用户在不离开 Twitter 的情境下对刚刷到的推文开价、交易，通过拥有某条推文的方式，支持相关的创作者。

赏金信息：

https://gitcoin.co/issue/DimensionDev/Maskbook/2599

- DeFi

⛓️ Mask Network x NEAR Protocol｜深度合作 & 联合赏金

Mask Network 与 PoS 底层公链 NEAR Protocol 达成深度合作，在技术以及应用等多层面进行交流。Mask Network 将有望接入其跨链工具彩虹桥（Rainbow Bridge）、EVM （以太坊虚拟机）、转账工具 NEAR Drop 等功能以完成 Mask Network 与 NEAR Protocol 的集成。

合作以发布联合赏金的方式启动，首个赏金的任务是将 NEAR Wallet 率先接入到 Mask Network 插件中，为未来进一步的合作奠定坚实基础。

赏金信息：

https://gitcoin.co/issue/DimensionDev/Maskbook/2602/100025151

🤝 Mask Network x Convergence｜预计实现 WSTs 的 ITO 发行

Mask Network 投资并与 DeFi 可互换资产协议 Convergence 合作，同时，Mask Network 创始人以及 CEO Suji Yan 也将成为 Convergence 的顾问之一。

Convergence 上的项目计划通过 Mask Network 特有的 ITO 功能发行 Wrapped Security Tokens （WSTs），且后续允许用户在 Convergence 的 AMM 池中进行交易。

🙌 Mask Network x Polygon｜探索更多玩法

Mask Network 与 Layer 2 解决方案聚合器 Polygon（原 Matic Network） 合作，双方针对以太坊目前出现的拥堵以及高矿工费问题展开技术交流协作。

Mask Network 将率先接入 Polygon 生态项目之一的去中心化交易所 QuickSwap，并继续通过发布赏金的形式逐步完成与 Polygon 的集成。

赏金信息：

https://gitcoin.co/issue/DimensionDev/Maskbook/2645/100025307

- 社区与开发

🎉 Mask Network x Gitcoin

Mask Network 从 Gitcoin 第 6 轮捐款活动开始就一直活跃在其资助行列，并陆续通过 Gitcoin 发布了 20 余则赏金；与此同时，在 Gitcoin 第 9 轮捐赠活动中 Mask Network 也继续提供了资金支持，鼓励更多开发者参与到未来区块链以及开源开发的生态建设中来。
